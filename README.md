# 💪 Work Out

## 소개

자신의 운동이나 일정을 기록하고 남들과 공유하여 동기부여를 얻는 서비스
자신의 기록을 그룹원들과 공유하여 동기부여를 얻어보세요.

**특징**

|     특징     |        설명         |
| :----------: | :-----------------: |
|   Next.js    |       13.0.3        |
|   상태관리   | React-Query, Recoil |
|    스타일    |  Styled-Component   |
| 데이터베이스 |      firebase       |

**깃 커밋 메시지 컨벤션**

- [feat]: 새로운 기능추가
- [bug]: 버그 수정
- [layout, style]: html, css 변경
- [refact, refactor]: 코드 리팩토링
- [doc]: .md 파일 수정
